# Security policy

FasalPramaan is currently a local, synthetic-data demonstration repository. It is not a public production service.

## Reporting a vulnerability

Do not include tokens, credentials, personal data, or exploit proof-of-concept details in a public issue. If the GitHub repository owner enables private vulnerability reporting, use that channel. Otherwise contact the repository owner privately through an agreed team channel.

## Scope

The repository includes application security controls for the demo, but it does not claim a penetration-test certification or production approval. See [docs/security.md](./docs/security.md) and [docs/production-readiness.md](./docs/production-readiness.md).
